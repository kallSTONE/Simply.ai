# SimplyAI

A hub for AI tools and resources, featuring categories, descriptions, and AI-powered functionality.

## Features

- Browse and filter AI tools
- Categories and tags
- Modern dark-themed UI
- Default AI avatar for tools

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Supabase (database & storage)
- GitHub for version control

## Getting Started

```bash
git clone https://github.com/kallSTONE/Simply.ai.git
cd SimplyAI
npm install
npm run dev
